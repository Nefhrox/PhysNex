Problem 2

Type: theory

Difficulty: 3/10

Text of a problem: Two spheres are suspended from a single point by two strings of equal length. Compare the angles of deviation of the strings from the vertical if:

1 The spheres have equal masses and carry like charges, with the charge of the first sphere greater than that of the second

2 The spheres have equal charges, but the mass of the first sphere is greater than that of the second

Answer:
The angles are equal

Solution:
\(\text{(1) } \pmb{T = mg} \)

\(\text{(2) } \pmb{T \sin{\alpha} = \frac{kq_1q_2}{r^2}} \)

\(\text{(3) } \pmb{\sin{\alpha} = \frac{kq_1q_2}{mgr^2}} \)

\(\text{(4) } \pmb{ \frac{\sin{\alpha_1}}{\sin{\alpha_2}}} = \frac{kq_1q_2}{m_1gr^2} \cdot \frac{m_2gr^2}{kq_1q_2} = \frac{m_2}{m_1}\)

\(\text{(5) } \pmb{m_1 = m_2} \text{ => } \frac{\sin{\alpha_1}}{\sin{\alpha_2}} = 1 \)

\(\text{(1) } \pmb{m_1 > m_2} \)

\(\text{(1) } \pmb{m_1 > m_2} \text{ => } \frac{\sin{\alpha_1}}{\sin{\alpha_2}} \text{ < } 1 \)
